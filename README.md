## Project Description
Memory Museum is a desktop memory card matching game developed using Python and Pygame. Players flip cards to find matching pairs. Every successful match restores a damaged museum painting. When all pairs are matched, the museum is fully restored and the player wins.

## Features
- Start Game
- Leaderboard
- Statistics
- Memory Card Matching
- Museum Restoration
- Score Tracking
- JSON Leaderboard Saving
- NumPy Statistics

## How to Run

1. Clone the repository.
2. Create a virtual environment.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run:

```bash
python main.py
```
